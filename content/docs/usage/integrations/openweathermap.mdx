---

OpenWeatherMap provides current weather conditions, forecasts, and air quality data. When enabled, weather information is shown on place detail cards.

---

## Capabilities

| Capability | Description |
|---|---|
| **Weather** | Current conditions, forecasts, and air quality for a location |

---

## Configuration

In **Settings → Integrations**, click the settings icon on the OpenWeatherMap card.

| Field | Value |
|---|---|
| **API Key** | Your OpenWeatherMap API key. Create one at [home.openweathermap.org/api_keys](https://home.openweathermap.org/api_keys). |

Click **Test Connection** before saving.

---

## Notes

OpenWeatherMap offers a free tier with up to 1,000 API calls/day, sufficient for most small deployments. See [openweathermap.org/price](https://openweathermap.org/price) for higher-volume plans. New API keys may take a few hours to activate.
