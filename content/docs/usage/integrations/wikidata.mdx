---
title: Wikidata
description: Structured place data and entity information via Wikidata.
---

Wikidata is a free, structured knowledge base maintained by the Wikimedia Foundation. The integration enriches place detail cards with additional structured data such as official website, social profiles, and historical information.

---

## Capabilities

| Capability | Description |
|---|---|
| **Place Info** | Supplements place records with Wikidata entity data |

---

## Configuration

In **Settings → Integrations**, click the settings icon on the Wikidata card.

No configuration is required — Wikidata is a fully public API with no authentication. Simply add the integration and enable it.

Click **Test Connection** before saving.
