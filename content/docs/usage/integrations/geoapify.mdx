Geoapify is a hosted mapping platform offering geocoding, place search, category browsing, and routing in a single API. It is a good all-in-one fallback option.

---

## Capabilities

| Capability | Description |
|---|---|
| **Search Category** | Browse places by category |
| **Autocomplete** | Address and place typeahead |
| **Geocoding** | Address-to-coordinate and reverse geocoding |
| **Place Info** | Fetch place detail by Geoapify place ID |
| **Routing** | Turn-by-turn directions |

### Supported route preferences

Avoid highways, avoid tolls, avoid ferries, safety vs. efficiency balance.

---

## Configuration

In **Settings → Integrations**, click the settings icon on the Geoapify card.

| Field | Value |
|---|---|
| **API Key** | Your Geoapify API key. Create one at [myprojects.geoapify.com](https://myprojects.geoapify.com). |

Click **Test Connection** before saving.

## Notes
