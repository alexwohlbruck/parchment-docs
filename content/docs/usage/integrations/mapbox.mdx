---
title: Mapbox
description: Interactive map tiles and base map rendering via Mapbox.
---

Mapbox provides the base map tiles rendered in Parchment's map view. It is the primary map engine integration.

---

## Capabilities

| Capability | Description |
|---|---|
| **Map Engine** | Vector and raster tile rendering for the main map |

---

## Configuration

In **Settings → Integrations**, click the settings icon on the Mapbox card.

| Field | Value |
|---|---|
| **API Key** | Your Mapbox public access token. Found in your [Mapbox account dashboard](https://account.mapbox.com) under **Access tokens**. Tokens start with `pk.`. |

Click **Test Connection** before saving.

---

## Notes

Mapbox tiles are subject to [Mapbox's pricing](https://www.mapbox.com/pricing) based on map load volume. The free tier covers development and low-traffic deployments.
