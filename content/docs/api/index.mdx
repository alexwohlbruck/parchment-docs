description: Reference for the Parchment REST API.
---

The Parchment API is a REST API that powers the Parchment map and trip-planning app. All responses are JSON.

In development the API is at `http://localhost:5000`. In production, use your instance’s base URL.

## Authentication

Most endpoints require authentication. Send credentials as a **session cookie** or an **API token** in the `Authorization: Bearer <token>` header. See [Auth](/docs/api/auth) for details.
