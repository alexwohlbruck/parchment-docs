## Generating the API Docs

Run the following command to regenerate these pages from your running server:

```bash
cd docs && bun run generate:api
```

With the shared `.env` at the monorepo root (so `SERVER_ORIGIN` is set):

```bash
cd docs && bun run generate:api
```

The script reads `SERVER_ORIGIN` from the environment (or falls back to `http://localhost:5000`) and fetches the spec from `{SERVER_ORIGIN}/docs/json`.
